## Portada
Para esta tarea se utilizó la portada de ejemplo del primer laboratorio. Sólo se cambió el título y se identificó el autor.

## Índice General
Para el archivo "IndiceContenido.tex" simplemente se colocó el comando \tableofcontents para desplegar automáticamente el índice general de la tarea.

## Índice de Figuras
Para el archivo "IndiceImagenes.tex" simplemente se colocó el comando \listoffigures para desplegar automáticamente el índice de aquellas figuras identificadas con la etiqueta {figure}.

## Contenido
Esta tarea está dividida por dos capítulos. El capítulo 1 comprende los conceptos básicos de de Big Data-Ciencia de los Datos y diversas áreas relacionadas; el capítulo 2 abarca la explicación de dos ecosistemas de Apache. Cada sección de la investigación fue identificada con la etiqueta \section{}. Los subtítulos dentro de los párrafos fueron identificados con la etiqueta \textbf{}, mientras que los subtítulos de cada sección fueron identificados con \subsection{}. Las herramientas de Apache Hadoop fueron seccionadas en viñetas (bullets) con la etiqueta \item. Las palabras en inglés dentro del contexto de cada párrafo se colocaron en cursiva con la etiqueta \emph{}.

## Bibliografía
En el archivo "Referencias.bib" se colocaron todas las referencias bibliográficas especificándoles un id, un título y un URL. Cada una de estas referencias está citada en el contenido temático con la etiqueta \cite{}.

## Carga de los archivos
Inicialmente, se clonó el directorio de la tarea con el comando:

git clone https://github.com/ICDRepository-I-2016/tarea-opcional-josemalvarezg1.git


Seguidamente, en el mismo directorio se agregaron manualmente los archivos "Tarea%20Opcional%201.zip" generado en ShareLaTeX en una carpeta "src", "Tarea_Opcional_1.pdf" en una carpeta "doc", y se editó este documento README.md.


Por último, se cargaron todos estos archivos con ayuda de los siguientes comandos:

git add .

git commit -m 'Cargando tarea opcional 1.'

git push
